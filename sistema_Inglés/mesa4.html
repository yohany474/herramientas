    • Condicional 0

    El condicional cero sirve para formar oraciones donde tanto la condición como el resultado son hechos reales. El mejor ejemplo de este tipo de oraciones son las relacionadas con argumentos científicos. Estructura gramatical del zero conditional:
    Opción 1: If + Condición (oración con verbo conjugado en present simple) + Resultado (oración con verbo conjugado en present simple)
    Opción 2: Resultado (oración con verbo conjugado en present simple) + if + Condición (oración con verbo conjugado en present simple).
    Ojo: Toma en cuenta que la posición de la condición y el resultado cambia de acuerdo a la posición de if.

    Ejemplos: If you mix red and blue, you get purple (Si mezclas rojo y azul, obtienes morado).
    An ice cube melts if you put in the sun (Un cubo de hielo se derrite si lo pones bajo el sol)

    Condicional 1

    El First Conditional es aquel que expresa una situación real, trayendo como consecuencia un posible resultado en el futuro que tiene altas probabilidades de que sea real.En inglés se usa para decir que un evento que puede suceder en el futuro:

    If + sujeto + presente simple + futuro (will) + verbo


    Como vemos en el ejemplo anterior, el uso de este condicional 1 o primer condicional en inglés es de lo básico que existe en este tema. Además, es aquel que más se usa en la gramática inglesa, ya que es el más simple y hace referencia a sucesos reales que puedan pasar.
    También, es importante destacar que el orden de la frase puede ser diferente, colocando el resultado al principio y la condición como lo que le sigue, veamos para mayor claridad:

    Sujeto + futuro (will) + verbo + complemento + if + sujeto + presente simple

    I ill learn more english if I practice every day / Yo aprenderé más inglés si practico todos los días


    Para este segundo caso, no es necesario agregar una coma con el fin de separar el condicional 1 ingles del posible resultado que se puede generar.
